# Agent Skills with Anthropic 交互式伴学资料

这是一个中文伴学版资料，基于 DeepLearning.AI 公开课程页和 Anthropic 官方 Skills 资料改写。公开页显示课程讲师为 Anthropic 的 Elie Schoppik；你可能是在吴恩达或 DeepLearning.AI 渠道看到它。它适合边看课程边做练习，尤其适合把 Agent Skills 的思路迁移到自己的 vibe coding 项目里。

## 打开方式

直接用浏览器打开：

```text
agent-skills-with-anthropic/index.html
```

如果通过 GitHub Pages 发布，访问：

```text
https://你的用户名.github.io/仓库名/agent-skills-with-anthropic/
```

## 内容范围

- 12 个学习单元：10 个视频章节、1 个材料阅读、1 个测验。
- 每章包含：目标、概念、例子、交互练习、小测验、可复用 AI 提示词和资料来源。
- 页面使用 `localStorage` 保存本地学习进度，不需要后端或构建步骤。
