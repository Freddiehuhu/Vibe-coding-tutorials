# Vibe Coding 基础交互式学习资料

这是一个可以直接在浏览器中打开的单页互动教程，面向刚开始用 AI 生成项目的学习者。

## 内容

- 变量、函数、对象、数组
- 条件判断、循环
- 模块导入导出
- async/await
- JSON
- 前端、后端与 API
- 数据库表
- 环境变量
- 综合项目生成练习

## 本地打开

直接打开 `index.html`。

## 发布到 GitHub Pages

把本仓库推送到 GitHub 后，在仓库页面进入：

`Settings -> Pages -> Build and deployment -> Deploy from a branch`

选择：

- Branch: `main`
- Folder: `/root`

保存后，页面地址通常是：

`https://你的用户名.github.io/仓库名/`
