# Vibe Coding 基础交互式学习资料

这是一个可以直接在浏览器中打开的单页互动教程，面向刚开始用 AI 生成项目的学习者，尤其适合只懂一点编程基础、还没有实际开发过产品的人。

学习方式不是先把所有语法背完，而是：

1. 选一个很小的产品。
2. 每次只做一个小功能。
3. 一边看概念，一边让 AI 帮你实现。
4. 自己运行、观察、复制报错、复盘。
5. 每完成一个小功能就提交 Git。

## 内容

### Vibe Coding 基础教程

- 新手实操路线
- 变量、函数、对象、数组
- 条件判断、循环
- 模块导入导出
- async/await
- JSON
- 前端、后端与 API
- 数据库表
- 环境变量
- 综合项目生成练习

### Agent Skills with Anthropic 伴学资料

- DeepLearning.AI x Anthropic 课程公开大纲梳理
- Skills、Tools、MCP server、Subagents 的区别
- 预置 Skills 拆解方法
- 自定义 SKILL.md 草稿练习
- Claude API、Claude Code、Agent SDK 的学习路径
- 每章交互练习、小测验和可复用提示词

## 本地打开

直接打开 `index.html`。

## 发布到 GitHub Pages

把本仓库推送到 GitHub 后，在仓库页面进入：

`Settings -> Pages -> Build and deployment -> Deploy from a branch`

选择：

- Branch: `main`
- Folder: `/root`

保存后，页面地址通常是：

`https://你的用户名.github.io/仓库名/`

本仓库里的 Agent Skills 伴学资料地址是：

`https://你的用户名.github.io/仓库名/agent-skills-with-anthropic/`
